# How to start with Scala

## Installation

### OSX

    brew install scala
    brew install sbt

## How to start a REPL

    $ scala

    Welcome to Scala 2.12.8 (Java HotSpot(TM) 64-Bit Server VM, Java 1.8.0_05).
    Type in expressions for evaluation. Or try :help.

    scala> var a : String = "a"
    a: String = a
